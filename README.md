# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning. It includes data preprocessing, exploratory data analysis, visualization, feature engineering, model training, evaluation, and an interactive prediction system.

---

## 🚀 Features

- Titanic Dataset Analysis
- Data Cleaning
- Missing Value Handling
- Data Visualization
- Feature Engineering
- Label Encoding
- Train-Test Split
- Random Forest Classifier
- Model Evaluation
- Feature Importance Analysis
- Interactive Survival Prediction
- Model Saving using Joblib

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## ▶️ How to Run

Clone Repository

```bash

```

Install Dependencies

```bash

```

```bash
python titanic_prediction.py
```
